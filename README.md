## Description

Fullstack music platform app. Nestjs on backend and React on frontend.

## Installation (in folders "client" and "server")

```bash
$ npm install
```

## Running server

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running client

First, run the development server:

```bash
npm run dev
```
